# Digital Business Card

Static Astro landing page for the live digital business card.

## Run locally

```bash
npm install
npm run dev
```

## Validate

```bash
npm run check
npm run build
```

## Environment variables

Copy `.env.example` to `.env` if you want to override the site URL:

```bash
PUBLIC_SITE_URL=https://your-domain.com
```

## Deploy

1. Push this folder to GitHub.
2. Connect the repository to Vercel.
3. Use `npm run build` as the build command.
4. Use `dist` as the output directory if Vercel asks for it.
